VICE is a program that runs on a Unix, MS-DOS, Win32, OS/2, Acorn RISC OS, BeOS, QNX 6.x, Amiga, GP2X, Dingoo, Syllable or Mac OSX machine and executes programs intended for the old 8-bit computers. The current version emulates the C64, the C64DTV, the C128, the VIC20, almost all PET models, the PLUS4 and the CBM-II (aka C610).

Download VICE for Mac OS X systems

You can download the latest VICE distributions for Mac OS X systems here. Some distributions may not be available at the release date.

  • Mac OS X Requirements:
    • On PowerPC Macs: Requires at least Mac OS X 10.3, Preferred: 10.5
    • On Intel Macs: Requires at least Mac OS X 10.4, Preferred: 10.5
    • X11.app Version 1.0 from Extra Installs on Mac OS X DVD for X11 and Gtk+ port.

  • Source distributions:
    • Tarball: vice-2.2.tar.gz

  • New binary distributions:
    • Universal Binary for X11 Athena Widgets UI: vice-macosx-x11-ub-10.3+4-2.2.dmg
    • Universal Binary for X11 Gtk+ UI: vice-macosx-gtk-ub-10.3+4-2.2.dmg
    • Universal Binary for native Cocoa UI (PowerPC>=10.3, Intel>=10.4): vice-macosx-cocoa-ub-10.3+4-2.2.dmg

      Note: The Cocoa Port is not complete yet. Although the archive contains x64.app and x64dtv.app with complete feature sets, all other emulators are still missing essential features. Please choose the X11/Athena or X11/Gtk+ port if you need features that are still missing in the Cocoa Mac release.

    • Universal Binary for native Cocoa UI (PowerPC>=10.5, Intel>=10.5): vice-macosx-cocoa-ub-10.5-2.2.dmg

      Note: This version requires at least Mac OS X 10.5 and offers extended features (e.g. the enhanced Resource Inspector).

  • Old binary distributions:
    • Universal Binary for X11 Athena Widgets UI: vice-macosx-x11-ub-2.1.dmg
    • Universal Binary for X11 Gtk+ UI: vice-macosx-gtk-ub-2.1.dmg
    • Universal Binary for native Cocoa UI: vice-macosx-cocoa-ub-2.1.dmg

      Note: The Cocoa Port is not complete yet. The archive contains a runnable x64 with a subset of the upcoming features. This binary is a technical preview of the new port and no full featured release of VICE. Please choose the X11/Athena or X11/Gtk+ port for a complete Mac release.
