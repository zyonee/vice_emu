12 Acknowledgments

VICE derives from X64, the first Commodore 64 emulator for the X Window System. Here is an informal list of the people who were mostly involved in the development of X64 and VICE:

The VICE core team:

  • Ettore Perazzoli (@email{ettore@comm2000.it}) made the 6510, VIC-II, VIC-I and CRTC emulations, part of the hardware-level 1541 emulation, speed optimizations, bug fixes, the event-driven cycle-exact engine, the Xt/Xaw/Xfwf-based GUI for X11, a general code reorganization, the new resource handling, most of the documentation. He also wrote the MS-DOS port and the initial MS-Windows port (well, somebody had to do it).
  • Daniel Sladic (@email{sladic@eecg.toronto.edu}) started the work on hardware-level 1541 emulation and wrote the new monitor introduced with VICE 0.15.
  • Andreas Boose (@email{boose@linux.rz.fh-hannover.de}) gave lots of information and bug reports about the VIC-II, the 6510 and the CIAs; moreover, he wrote several test-routines that were used to improve the emulation. He also added cartridge support and has been the main head behind the drive and datasette emulation since version 0.15. Also added several UI elements to the MSDOS and MS-Windows ports and the C128 Z80 implementation.
  • Dag Lem (@email{resid@nimrod.no}) implemented the reSID SID emulation engine.
  • Tibor Biczo (@email{crown@mail.matav.hu}) improved the MS-Windows port.
  • Andreas Dehmel (@email{dehmel@forwiss.tu-muenchen.de}) wrote the Acorn RISC OS port.
  • Thomas Bretz (@email{tbretz@ph.tum.de}) responsible for the OS/2 port.
  • Andreas Matthies (@email{andreas.matthies@arcormail.de}) improved the datasette support and some ui stuff in the Win32 and MSDOS port. Wrote the BeOS port.
  • Martin Pottendorfer (@email{Martin.Pottendorfer@aut.alcatel.at}) Implemented Gnome Port based on Oliver Schaertels GTK+ port Added support code for internationalization based on gettext Translated the Unix Port to German Implemented the fliplists + ui (unix).
  • Markus Brenner (@email{markus@brenner.de}) added VDC emulation to x128.

Former team members:

  • Andre Fachat (@email{fachat@physik.tu-chemnitz.de}) wrote the PET and CBM-II emulators, the CIA and VIA emulation, the IEEE488 interface, implemented the IEC serial bus in `xvic' and made tons of bug fixes.
  • Teemu Rantanen (@email{tvr@cs.hut.fi}) implemented the SID emulation and the trap-based disk drive and serial bus implementation; added support for multiple display depths under X11. Also wrote `c1541'
  • Jouko Valta (@email{jopi@stekt.oulu.fi}) wrote petcat and c1541, T64 handling, user service and maintenance (most of the work in x64 0.3.x was made by him); retired from the project in July 96, after VICE 0.10.0.
  • Jarkko Sonninen (@email{sonninen@lut.fi}) was the founder of the project, wrote the old version of the 6502 emulation and the XDebugger, and retired from the project after x64 0.2.1.
